Enunciado

Para estudiar el efecto de distintos fertilizantes se observ el crecimiento del tallo, en un perodo determinado, en plantas de menta cultivadas en parcelas de terreno tratadas con diferentes soluciones nutritivas. Interesa verificar si con los fertilizantes estudiados se obtiene un crecimiento medio mayor que con el utilizado como control. Los datos obtenidos, en cm, son los siguientes:

Datos

Armamos el data frame con los datos

crecimiento = c(2.70,1.72,3.24,4.28,4.20,4.73,4.05,2.58,4.87,6.41,6.30,7.10,11.10,8.17,12.73,15.83,15.60,17.20,16.42,10.31,18.53,22.95,23.67,25.34,9.75,7.31,11.11,13.69,13.50,14.83)
fertilizante = factor(rep(c("Control","A","B","C","D"),each=6),levels = c("Control","A","B","C","D"))
tabla.ANOVA=data.frame(crecimiento,fertilizante)
head(tabla.ANOVA)

Prestar atencin al parmetro levels en el factor fertilizante ya que es importante el orden y se explicar el motivo ms adelante.

Normalidad

Realizamos el test correspondiente de normalidad de las muestras. Nuestras hiptesis son

\[ \left\{ \begin{array}{ll} H_0: \mbox{ La distribucin de $X_i$ es del tipo $N(\mu,\sigma)$, $\forall i=1, ..., 5$}\\ H_1: \mbox{ La distribucin de $X_i$ no es del tipo $N(\mu,\sigma)$, $\forall i=1, ..., 5$} \end{array} \right. \]

for (f in unique(fertilizante)) {
  norm.test = shapiro.test(tabla.ANOVA[tabla.ANOVA$fertilizante==f,1])
  
  if(norm.test$p.value < 0.05){
    print(norm.test)
  }
}

Vemos que las diferentes subpoblaciones siguen una distribucin normal.

Homocedasticidad

Realizamos el test correspondiente de homocedasticidad de las muestras. Nuestras hiptesis son

\[ \left\{ \begin{array}{l} H_0 : \sigma_1^2 =\sigma_2^2 =\cdots =\sigma_k^2, \mbox{con $k=5$} \\ H_1 : \exists i,j \mid \sigma_i^2 \not=\sigma_j^2. \end{array} \right. \]

bartlett.test(crecimiento ~ fertilizante,data=tabla.ANOVA)
## 
##  Bartlett test of homogeneity of variances
## 
## data:  crecimiento by fertilizante
## Bartlett's K-squared = 12.563, df = 4, p-value = 0.01362

Puesto que se rechaza \(H_0\), no podemos suponer que las varianzas son iguales.

Trataremos de encontrar una transformacin de manera que, con los datos transformados, las varianzas resulten homogneas.

Para eso veremos si existe alguna relacin entre la media y la desviacin estndar de las muestras.

boxplot(crecimiento ~ fertilizante,data=tabla.ANOVA)

En el grfico podemos observar que, para los fertilizantes B y C que presentan un crecimiento mayor, la dispersin de los datos tambin es mayor, mientras que para el Control el crecimiento es menor y tambin es menor la dispersin.

Esto nos sugiere que la desviacin estndar de las muestras aumenta a medida que aumenta la media.

Si hay una relacin de proporcionalidad entre la media y la desviacin estndar se puede lograr homogeneizar las varianzas aplicando a los datos la transformacin logartmica.

Esa relacin lineal la observamos en el grfico de la desviacin estndar en funcin de la media, por eso transformamos los datos mediante el logaritmo obteniendo la siguiente tabla

Realizamos dicha transformacin logartmica sobre los datos en la columna crecimiento de la tabla del ANOVA que construimos al principio

tabla.ANOVA$crecimiento = log10(tabla.ANOVA$crecimiento)

Realizamos nuevamente el test de homocedasticidad

bartlett.test(crecimiento ~ fertilizante,data=tabla.ANOVA)
## 
##  Bartlett test of homogeneity of variances
## 
## data:  crecimiento by fertilizante
## Bartlett's K-squared = 1.0009, df = 4, p-value = 0.9097

Esta vez, no podemos rechazar \(H_0\) y aceptamos la homogeneidad de varianzas.

ANOVA

Con las variables transformadas efectuamos el ANOVA. Nuestras hiptesis son

\[ \left\{ \begin{array}{l} H_0 : \mu_1 =\mu_2 =\cdots =\mu_k, \mbox{con $k=5$}\\ H_1 : \exists i,j \mid \mu_i \not=\mu_j. \end{array} \right. \]

resultado.anova = aov(crecimiento ~ fertilizante,data=tabla.ANOVA)

summary(resultado.anova)

Dado el p-valor obtenido, concluimos que hay diferencias significativas entre los crecimientos medios del tallo obtenidos con los distintos fertilizantes.

El objetivo de la experiencia es verificar si con los fertilizantes analizados se obtiene un crecimiento medio mayor que el obtenido con el control. Nuestras hiptesis son

\[ \left\{ \begin{array}{l} H_0 : \mu_i = \mu_1, \mbox{$\forall i=1, ..., 5$}\\ H_1 : \mu_i > \mu_1, \mbox{$\forall i=1, ..., 5$}. \end{array} \right. \]

Vamos a utilizar dos test a modo de ejemplo para realizar esta comparacin por parejas pero antes una aclaracin. Se acuerdan que les dije al principio que recordaran el parmetro levels en el factor fertilizante? Pues bien, el nivel del cual se quiere realizar la comparacin de menor o mayor que el resto de las medias debe figurar primero. Es por ese motivo que especificamos con el parmetro levels el orden de los niveles del factor, y en este caso como queremos comparar cual de todos los fertilizantes tiene una media mayor que el control, ponemos al nivel Control primero en la lista de niveles del factor.

Utilizaremos la funcin pairwise.t.test para realizar la comparacin por parejas y adems la funcin dunnettTest del paquete PMCMRplus que es muy utilizada en los casos donde justamente se quiere hacer una comparacin contra una muestra de control. Esta ltima funcin utiliza para la comparacin una distribucin llamada distribucin t de Dunnett

library(PMCMRplus)

pairwise.t.test(tabla.ANOVA$crecimiento,
                tabla.ANOVA$fertilizante,p.adjust.method = "holm",alternative="greater")

Vamos analizar los resultados obtenidos.

Dado los p-valores obtenidos en ambas tablas, con nivel de significacin del 5% concluimos que:

  • Con los fertilizantes B, C y D se obtiene un crecimiento medio del tallo significativamente mayor que el obtenido con el Control.

  • En el caso del fertilizante A, estamos en la zona de penumbra pero podemos realizar un segundo test, esta vez de dos colas para ver el resultado obtenido. Nuestras hiptesis son
